## Features

- **Live Match Statistics** — Real-time scores, events, and stats for 50+ leagues
- **AI Pre-Match Analysis** — Poisson distribution model with Gemini AI commentary
- **Live AI Predictions** — In-play corner, goal, card, and foul predictions
- **1X2 Match Result** — Win/draw/loss probability with 11x11 Poisson matrix
- **Smart Odds Tracking** — Live odds comparison, minimum 1.35 threshold
- **Self-Learning AI** — Weekly learning engine optimizes prediction rules automatically
- **User Predictions** — Make your own predictions and compete on the leaderboard

## How It Works

1. **Data Collection** — Season-long team statistics collected via API-Football
2. **Poisson Model** — Home/away goal projections calculated per organization (league/cup/CL)
3. **AI Analysis** — Gemini 2.5 Flash generates match commentary and insights
4. **Live Predictions** — Real-time stats analyzed every minute during live matches
5. **Learning Loop** — Weekly performance review adjusts thresholds and rules

## Tech Stack

- **Frontend:** HTML/CSS/JS (Web), Flutter/Dart (iOS & Android)
- **Backend:** Supabase (PostgreSQL + Edge Functions + Realtime)
- **AI:** Google Gemini 2.5 Flash, Groq Llama 3.3
- **APIs:** API-Football for live data
